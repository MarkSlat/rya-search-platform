## RYA Search Platform
### Overview
This is a Flask-based web application to manage flights and airports. The app allows you to view trips, visualise flight routes on a map, and manage base airports through an admin panel.

### Dependencies
- Flask
- Bootstrap 4
- Leaflet.js
- Neo4j Python Driver
