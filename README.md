## RYA Search Platform
### Overview
This is a Flask-based web application to manage flights and airports. The app allows you to view trips, visualise flight routes on a map, and manage base airports through an admin panel.

### To Run
### Build image
```shell
docker compose build
```
or
```shell
podman-compose build
```
### Run
```shell
docker compose up -d
```
or
```shell
podman-compose up -d
```

### Stop
```shell
docker compose down
```
or
```shell
podman-compose down
    ```

### Dependencies
- Flask
- Bootstrap 4
- Leaflet.js
- Neo4j Python Driver
